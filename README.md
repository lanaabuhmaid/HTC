# Hashemite Training Center 

## Description
This project is a web application built using [nodeJS / Express.js / PostgreSQL / JavaScript / etc].  
The Hashemite Training Center is designed to bridge the gap between academic
learning and real-world professional experience by providing field training
resources and support for students. This platform addresses the challenges
students face in finding suitable training opportunities aligned with their
specialties, GPA, and geographical preferences. Utilizing a user-centered design,
the website offers features that allow students to sort companies by relevant
criteria, submit CVs directly to potential employers, and access customizable CV
templates. Furthermore, students can explore both free and paid courses, receive
personalized notifications, and track their progress through a dedicated task log,
which is evaluated annually. The platform also includes a Q&A section that guides
students on the application process and highlights necessary qualifications. The
project aims to simplify and streamline the field training experience, ultimately
empowering students to build the skills and confidence required to transition
smoothly into their careers.
